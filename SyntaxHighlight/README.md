<h1>Syntax highlighter for UEStudio/UltraEdit</h1>

- ABB RAPID IRC5
- Functionlist treeview of procs+arguments, funcs+arguments, traps
- Highlights:
-   RAPID instructions and functions
-   RAPID datatypes
-   RAPID built-in default variables
-   argument switches
-   error constants
