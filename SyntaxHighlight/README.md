<h1>Syntax highlighter for UEStudio/UltraEdit</h1>

<h3>Features</h3>
- ABB RAPID IRC5

<h5>Functionlist</h5>
- module name
- procedures (global/local) + arguments
- functions (global/local) + arguments
- traps (global/local)
- records
- workobjects
- tooldata
- syntax errors

<h5>Highlighting</h5>
- RAPID instructions and functions
- RAPID datatypes
- RAPID built-in default variables
- argument switches
- error constants
